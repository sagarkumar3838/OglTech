Method 1: Automatic Detection

Method 2: Manual Check

If automatic detection doesn't work:

  1. Make sure you're logged in to http://localhost:3000
  2. Open browser console (Press F12)
  3. Type: localStorage.getItem('user')
  4. Look for the "id" field in the output

Method 3: From Supabase

Run this query in your Supabase SQL Editor:

SELECT DISTINCT user_id FROM user_progress LIMIT 1;

Next Steps

Once you have your User ID:

  1. Open seed-sample-dashboard-data.sql
  2. Replace 'YOUR_USER_ID' with your actual user ID
  3. Run the SQL in Supabase SQL Editor
  4. Go to Dashboard to see the results
